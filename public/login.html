Comment configurer Firebase

  1. Créez un projet sur console.firebase.google.com
  2. Activez Authentication → Google et Email/Mot de passe
  3. Dans Paramètres du projet → Vos apps → SDK configuration
  4. Ajoutez les secrets dans Replit (onglet Secrets) :
    FIREBASE_API_KEY, FIREBASE_AUTH_DOMAIN,
    FIREBASE_PROJECT_ID, FIREBASE_APP_ID
  5. Relancez l'application
